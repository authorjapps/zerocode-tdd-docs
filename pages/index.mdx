# Introduction

Welcome! To The documentation site for [zerocode-tdd](https://github.com/authorjapps/zerocode).

## What is Zerocode TDD?

Zerocode TDD is Super simple testing framework suitable for fast and repeatable builds with an instantly familiar syntax using JSON or YAML.

It is community-developed, free, opensource, automated testing framework built for Microservices APIs, Kafka, Database, Security and Load testing.

Zerocode TDD enables you to create, change, orchestrate and maintain your automated test scenarios declaratively

## Quick Start

Download or close the "helloworld" repo and run the sample tests locally

## Local Development

First, run `mvn clean install -DskipTests` to install the dependencies.

Then, run `http-testing` module tests

or

run `kafka-testing` module tests
